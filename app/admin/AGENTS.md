# AGENTS.md - Admin Vistaire

- `/admin` est un apercu restaurateur de demonstration, leger et public, pas un vrai admin production ouvert.
- L'experience doit soutenir la valeur du menu, pas devenir un dashboard SaaS froid.
- Ne pas inventer ventes, revenus, ROI, commandes, satisfaction ou conversion.
- Les insights doivent rester anonymes, comportementaux et fondes sur les donnees disponibles.
- Ne pas ajouter auth/backend lourd sans demande explicite.
- Ne pas exposer donnees production, secrets ou appels IA couteux sans protection, rate limit ou re-scope demo explicite.
- Tout changement UI admin exige responsive mobile, console propre et etats loading/error si necessaires.
